# Freelancer Operating System

Documentazione operativa per evolvere il pannello admin Caldes in un sistema completo per freelancer creativo.

Obiettivo centrale:

`lead -> preventivo -> approvazione -> progetto -> produzione -> revisione -> consegna -> pagamento -> retention`

## Struttura

- `01-master-plan.md`
Descrive visione, priorita, fasi e metriche.
- `02-functional-architecture.md`
Descrive menu admin target, verticali servizio e flussi operativi.
- `03-technical-schema.md`
Descrive schema dati, API e regole tecniche.
- `04-delivery-tracker.md`
Tracker di implementazione con checklist e stato avanzamento.

## Come usarlo durante realizzazione

1. Allineare scope e priorita su `01-master-plan.md`.
2. Validare UX e processi su `02-functional-architecture.md`.
3. Implementare migration e route usando `03-technical-schema.md`.
4. Tracciare avanzamento reale in `04-delivery-tracker.md`.
