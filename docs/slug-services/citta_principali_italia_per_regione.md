# Città principali italiane divise per regione

Scelta adottata per avere una base SEO pulita e difendibile:
**capoluoghi di provincia / città metropolitane**, con il **capoluogo di regione** messo per primo.

Pattern consigliato slug:
`/sito-web-per-{professione}-a-{citta}`
