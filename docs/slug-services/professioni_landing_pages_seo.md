# Professioni / categorie per landing page

Pattern consigliato slug:
`/sito-web-per-{professione}-a-{localita}`

Esempio:
`/sito-web-per-parrucchieri-a-frosinone`

## Regole rapide
- usa il plurale quando la query è naturalmente cercata al plurale
- mantieni una sola forma per categoria
- evita sinonimi duplicati in più pagine quasi uguali
