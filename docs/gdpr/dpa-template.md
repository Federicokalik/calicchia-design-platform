## Contenuto minimo del DPA (Art. 28(3))

Un DPA valido deve includere:

### 1. Oggetto e durata
- Descrizione dei servizi forniti
- Durata del trattamento

### 2. Natura e finalità del trattamento
- Quali operazioni vengono svolte sui dati
- Perché vengono trattati

### 3. Tipi di dati personali
- Categorie di dati trattati (es. email, nome, IP)

### 4. Categorie di interessati
- Chi sono le persone i cui dati vengono trattati

### 5. Obblighi del responsabile (sub-responsabile)
- [ ] Trattare i dati solo su istruzioni documentate del titolare
- [ ] Garantire che le persone autorizzate si siano impegnate alla riservatezza
- [ ] Adottare misure di sicurezza adeguate (art. 32)
- [ ] Non ricorrere a un altro responsabile senza autorizzazione scritta
- [ ] Assistere il titolare per le richieste degli interessati
- [ ] Assistere il titolare per la conformità agli artt. 32-36
- [ ] Cancellare o restituire tutti i dati al termine del rapporto
- [ ] Mettere a disposizione tutte le informazioni per dimostrare conformità
- [ ] Notificare le violazioni senza ingiustificato ritardo

### 6. Sub-responsabili
- Elenco dei sub-responsabili autorizzati
- Procedura di notifica per nuovi sub-responsabili

### 7. Trasferimenti extra-UE
- Garanzie adottate (SCCs, DPF, decisione di adeguatezza)

### 8. Audit
- Diritto del titolare di effettuare audit o ispezioni
